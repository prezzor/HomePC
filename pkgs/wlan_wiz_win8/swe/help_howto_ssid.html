Var finns mitt SSID?
Information om SSID kan finnas i dokumentationen från det företag som tillverkat din trådlösa åtkomstpunkt (router).

Information om SSID kan eventuellt finnas på förseglingen på din trådlösa åtkomstpunkt (router).



Om du har frågor om den här informationen ber vi dig kontakta systemadministratören, Internetleverantören eller det företag som tillverkat din trådlösa åtkomstpunkt (router).
