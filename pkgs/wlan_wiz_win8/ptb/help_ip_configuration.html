Configuração do endereço IP
A impressora/aparelho multifunção sem fios deve ser configurada para usar um endereço IP. Há duas opções possíveis:

Obter um endereço IP automaticamente. Usar esta opção se sua rede estiver utilizando um servidor DHCP.

Definir endereço IP estático. Use esta opção se sua rede não estiver utilizando um servidor DHCP e você deseja inserir manualmente o endereço IP e as configurações de máscara de sub-rede.

Se você não estiver certo qual opção selecionar, fale com seu administrador de sistema.

Observação: se você selecionar “Obter um endereço IP automaticamente”, sua impressora/aparelho multifunção tentará adquirir um endereço IP de um servidor DHCP. Se não for possível se comunicar com o servidor DHCP porque este se encontra indisponível ou você definiu incorretamente as configurações de segurança sem fios, a impressora/aparelho multifunção passará para um endereço IP baseado em APIPA na faixa 169.254.0.1 a 169.254.255.254.
