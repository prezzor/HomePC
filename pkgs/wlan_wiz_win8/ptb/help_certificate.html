Configuração do Certificado.
Certificado do Cliente
Selecione o certificado do cliente para autenticação EAP-TLS. É necessário configurar primeiro o Certificado na página "Configurar Certificado" da página da Web interna (EWS) do dispositivo.(Não é necessário selecionar o certificado do cliente ao utilizar um método de autenticação, exceto o EAP-TLS.)
Verificação do Certificado do Servidor
Selecione o método de verificação do certificado do servidor.(Não será preciso selecionar o método de verificação quando usar LEAP.)
  • Sem Verificação
    O certificado do servidor é confiável sem verificação.

  • Cert. CA
    O certificado do servidor emitido pelo CA é verificado pelo certificado de CA instalado neste dispositivo.É necessário configurar primeiro o Certificado na página "Configurar Certificado" da página da Web interna (EWS) do dispositivo.

  • Cert. CA + ID do Servidor
    O Certificado do Servidor (emitido pelo CA através do Certificado de CA instalado nesta máquina) e o Nome Comum são verificados.É necessário configurar primeiro o Certificado na página "Configurar Certificado" da página da Web interna (EWS) do dispositivo.

ID do Servidor
Insira a ID do servidor para a verificação do certificado do servidor.A ID do servidor é o nome comum do certificado do servidor.

Configurar Certificado
Se estiver utilizando um Certificado do Cliente ou um Certificado de CA, você precisará configurar primeiro o Certificado na página "Configurar Certificado" da página da Web interna (EWS) do dispositivo.
