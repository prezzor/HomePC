Appareils sans fil disponibles
Sélectionnez l'imprimante/appareil sans fil approprié puis cliquez sur “Suivant”.

Si votre périphérique d’impression sans fil n’est pas répertorié, cliquez sur le bouton “Actualiser” pour tenter de relancer une recherche dans votre environnement. Si vous ne le voyez toujours pas, veuillez effectuer les vérifications suivantes :
  1. Vérifiez que votre imprimante/appareil multifonction est allumé.

  2. (Uniquement applicable aux utilisateurs qui n’emploient pas de câble USB ou LAN pour configurer leur périphérique d’impression sans fil). Essayez de rapprocher votre périphérique d’impression sans fil de votre PC ou du point d’accès sans fil (selon le type de connexion).

  3. Vous pouvez temporairement désactiver tout programme de pare-feu s’exécutant sur votre ordinateur. En effet, il peut empêcher votre ordinateur de détecter les périphériques d’impression sans fil. Si vous choisissez de désactiver temporairement votre pare-feu, il est recommandé de déconnecter votre point d’accès de l’Internet pour éviter les failles de sécurité dans votre ordinateur.
