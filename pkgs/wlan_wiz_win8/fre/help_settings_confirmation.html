Confirmation du paramétrage du réseau sans fil
Pour confirmer tous les paramètres sans fil, cliquez sur Suivant.

Si vous êtes satisfait des paramètres sans fil de l'imprimante/appareil multifonctions mais que vous souhaitez changer l'adresse IP, cliquez sur “Modifier l'adresse IP”. Vous serez ainsi autorisé à changer les paramètres d'adresse IP de l'appareil sans changer les options de sécurité sans fil.
