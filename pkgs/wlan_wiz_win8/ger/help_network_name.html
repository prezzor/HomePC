Name des Wireless-Netzwerkes
Geben Sie dem Wireless-Gerät (Drucker/Multifunktionsgerät) einen Netzwerknamen (SSID), mit dem das Gerät verbunden werden soll.

Wird das Gerät in einem Ad-hoc-Netzwerk ohne Access Point genutzt, aktivieren Sie bitte die Option „Dies ist ein Ad-hoc-Netzwerk und es gibt keinen Access Point/Basisstation“. Ist diese Option aktiviert, müssen Sie den verwendeten Netzwerkkanal manuell eingeben. Falls Sie nicht Ihren verwendeten Netzwerkkanal wissen, dann entnehmen Sie bitte der dem Computer mitgelieferten Dokumentation wie Kanäle eingestellt werden.

Eine Personal Firewall schützt Ihren Computer vor unliebsamen Angriffen. Möglicherweise verhindert die aktive Firewall-Software auf Ihrem Computer das Erkennen des Wireless-Netzwerkes. Deaktivieren Sie in diesem Fall die Firewall-Software vorübergehend.

Wenn Sie Ihre Firewall-Software deaktivieren, ist Ihr Computer so lange böswilligen Angriffen ausgesetzt, bis Sie die Firewall wieder aktivieren. Wir empfehlen daher, den Access Point vom Internet zu trennen, bevor Sie die Firewall deaktivieren. Trennen Sie hierfür den Access Point vom Ethernet-Hub/Switch oder vom DSL- bzw. Kabelmodem, das die Internetverbindung herstellt.
