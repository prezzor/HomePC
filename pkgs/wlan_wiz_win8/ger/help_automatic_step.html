Auto-Wireless-Setup (per Tastendruck) verwenden
Die Option Auto-Wireless-Setup erlaubt diesem Drucker-/Multifunktionsgerät, seine Einstellungen zur Wireless-Netzwerkkonfiguration direkt von einem Wireless Access Point zu erhalten, der Wi-Fi Protected SetupTM oder AOSSTM unterstützt.
Wenn während der Einrichtung ein Problem auftritt, sollten Sie Ihr Gerät vorübergehend näher am Wireless Access Point aufstellen.
Weitere Informationen darüber, wie Sie Ihren Wireless Access Point so konfigurieren, dass er mit Wi-Fi Protected SetupTM und AOSSTM arbeitet, finden Sie im Benutzerhandbuch des Herstellers.
Wenn die Konfiguration Ihres Drucker-/Multifunktionsgerätes abgeschlossen ist, können Sie es zurück an einen geeigneten Ort in Ihrer Arbeitsumgebung stellen, der sich innerhalb der Reichweite des Wireless Access Points befindet.
