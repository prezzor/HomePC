Verfügbare Wireless-Netzwerke
Wenn Sie in einer Umgebung mit mehreren drahtlosen Netzwerken arbeiten, werden möglicherweise verschiedene drahtlose Netzwerke angezeigt. Wählen Sie in diesem Fall das entsprechende Netzwerk aus und klicken Sie dann auf „Weiter“. Falls Sie nicht wissen, welches das richtige Netzwerk ist, wenden Sie sich bitte an Ihren Systemadministrator.

Falls Ihr Wireless-Netzwerk nicht angezeigt wird, klicken Sie bitte auf die Schaltfläche „Aktualisieren“, um die Liste der aufgeführten Netzwerke auf den aktuellen Stand zu bringen. Wenn danach Ihr Wireless-Netzwerk immer noch nicht angezeigt wird, Sie aber sicher sind, dass der Netzwerkname (SSID) nicht ausgeblendet ist, dann prüfen Sie bitte Folgendes:
  1. Stellen Sie sicher, dass Ihr Wireless Drucker/DCP/MFC angeschaltet ist.

  2. Versuchen Sie, das Wireless-Gerät (Drucker/Multifunktionsgerät) dichter an den Access Point oder Computer zu stellen.

  3. (Nur für Nutzer, die kein USB-Kabel zur Konfiguration ihres Wireless-Druckgerätes verwenden.) Weil eine aktive Firewall-Software auf Ihrem Computer eventuell das Erkennen der Wireless-Netzwerke verhindert, sollten Sie vielleicht die Firewall-Software vorübergehend deaktivieren. Falls Sie die Firewall vorübergehend deaktivieren, sollten Sie ggf. Ihren Access Point vom Internet trennen, um Sicherheitslücken auf Ihrem Computer zu vermeiden.
Wenn Sie Ihren Access Point so konfiguriert haben, dass er die SSID unterdrückt, müssen Sie den Netzwerknamen manuell eingeben, indem Sie auf die Schaltfläche "Erweitert" klicken.

Hinweis: Die Wireless Signalstärke ist die Signalstärke zwischen dem Wireless-Gerät (Drucker/Multifunktionsgerät) und dem Access Point, mit dem er kommuniziert. Es ist nicht die Signalstärke zwischen Ihrem Computer und dem Access Point.
