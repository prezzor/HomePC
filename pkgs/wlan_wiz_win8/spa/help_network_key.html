Configuración de la clave de red
Escriba la clave de red adecuada y haga clic en “Siguiente”. Cada método de seguridad tiene sus propias normas.
Sistema abierto/clave compartida con WEP
Esta clave es un valor de 64 ó 128 bits en formato ASCII o hexadecimal.
  • 64 (40) bits ASCII:
    Usa 5 caracteres de texto. Ejemplo: “WSLAN” (distingue entre mayúsculas y minúsculas).
  • 64 (40) bits hexadecimal:
    Usa 10 dígitos de datos hexadecimales. Ejemplo: “71f2234aba”.
  • 128 (104) bits ASCII:
    Usa 13 caracteres de texto. Ejemplo: “Wirelesscomms” (distingue entre mayúsculas y minúsculas).
  • 128 (104) bits hexadecimal:
    Usa 26 dígitos de datos hexadecimales. Ejemplo: “71f2234ab56cd709e5412aa2ba”.
WPA-PSK/WPA2-PSK y TKIP o AES
Usa una clave previamente compartida (PSK), que tiene entre 8 y 63 caracteres.
