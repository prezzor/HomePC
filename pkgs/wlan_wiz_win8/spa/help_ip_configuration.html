Configuración de la dirección IP
La impresora/dispositivo multifunción inalámbrico debe estar configurado para utilizar una dirección IP. Existen dos posibles opciones:

Obtener una dirección IP automáticamente. Utilice esta opción si su red está utilizando un servidor DHCP.

Establecer una dirección IP estática. Utilice esta opción si su red no está utilizando un servidor DHCP y desea introducir manualmente los valores de dirección IP y máscara de subred.

Si no está seguro de qué opción seleccionar, consulte a su administrador del sistema.

Nota informativa: si selecciona la opción “Obtener una dirección IP automáticamente”, su impresora/dispositivo multifunción intentará obtener una dirección IP de un servidor DHCP. Si no puede comunicar con el servidor DHCP debido a que el servidor DHCP no está disponible o ha configurado incorrectamente las opciones de seguridad inalámbrica, la impresora/dispositivo multifunción cambiará a una dirección IP con base APIPA en el intervalo 169.254.0.1 a 169.254.255.254.
