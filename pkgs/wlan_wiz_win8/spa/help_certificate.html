Configuración de certificado.
Certificado de cliente
Seleccione el certificado de cliente para autenticación EAP-TLS. Debe tener en cuenta que primero necesita configurar el certificado en la página "Configurar certificado" desde la página web interna (EWS) de su dispositivo. (No tiene que seleccionar el certificado de cliente si usa un método de autenticación, a excepción de EAP-TLS).
Verificación del certificado del servidor
Seleccione el método de verificación del certificado del servidor.(No tiene que seleccionarlo si usa LEAP.)
  • No verificar
    Se confía en el certificado del servidor sin verificarlo.

  • Cert. CA
    El certificado del servidor emitido por CA es verificado por el certificado CA instalado en este dispositivo.Debe tener en cuenta que primero necesita configurar el certificado en la página "Configurar certificado" desde la página web interna (EWS) de su dispositivo.

  • Cert. CA + ID servidor
    El certificado del servidor (emitido por CA utilizando un certificado CA instalado en este equipo) y el nombre común (ID del servidor) están verificados.Debe tener en cuenta que primero necesita configurar el certificado en la página "Configurar certificado" desde la página web interna (EWS) de su dispositivo.

ID del servidor
Escriba el ID del servidor para la verificación del certificado del servidor. El ID del servidor es el nombre común del certificado del servidor.

Configurar certificado
Si utiliza un certificado de cliente o certificado CA, primero debe configurar el certificado en la página "Configurar certificado" desde la página web interna (EWS) de su dispositivo.
