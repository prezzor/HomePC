Où est ma clé de réseau ?
Vous pouvez trouver des informations sur votre Clé de sécurité réseau dans la documentation fournie par le fabricant de votre point d'accès sans fil.

Il est possible que vous trouviez votre Clé de sécurité réseau à partir du sceau figurant sur votre point d'accès sans fil/routeur.



Si vous n'êtes pas sûr de ces informations, consultez votre administrateur système, votre fournisseur d'accès Internet ou le fabricant de votre point d'accès sans fil.
