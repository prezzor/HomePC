Nom du réseau sans fil
Entrez le nom de réseau (SSID) auquel sera associé l'imprimante/appareil multifonction.

Si l'appareil communique avec un réseau ad-hoc et qu'il n'y a pas de point d'accès, activez l'option “Il s’agit d’un réseau ad-hoc, sans point d’accès.”. Si vous sélectionnez cette option, vous devez aussi sélectionner manuellement le canal de réseau utilisé. Si vous ne savez quel canal de réseau vous utilisez, consultez la documentation accompagnant votre ordinateur et cherchez les informations sur les paramètres des canaux.

Un pare-feu personnel protège votre ordinateur des attaques malveillantes. Si vous utilisez un pare-feu personnel sur votre ordinateur et qu'il vous empêche de détecter les réseaux sans fil ou les appareils sur le réseau sans fil, nous vous recommandons de désactiver temporairement le pare-feu.

Cependant, en désactivant le pare-feu, vous exposez votre ordinateur aux attaques jusqu'à ce que vous le réactiviez. C'est pourquoi nous vous recommandons de déconnecter votre point d'accès de l'Internet avant de désactiver le pare-feu. Pour ce faire, déconnectez votre point d'accès du hub/commutateur Ethernet auquel il est connecté ou déconnectez le point d'accès du modem DSL ou câblé assurant votre connexion Internet.
