Configuração do endereço IP
A impressora/dispositivo multifunções sem fios tem de ser configurada(o) para utilizar um endereço. Tem duas opções possíveis:

Obter um endereço IP automaticamente. Utilize esta opção se a sua rede estiver a utilizar um servidor DHCP.

Definir endereço IP estático Utilize esta opção se a sua rede não estiver a utilizar um servidor DHCP e quiser introduzir manualmente as definições de endereço IP e de máscara de sub-rede.

Se não tiver a certeza qual a opção que deve seleccionar, contacte o seu administrador de sistema.

Nota: se seleccionar a opção "Obter um endereço IP automaticamente", a sua impressora/dispositivo multifunções tentará adquirir um endereço IP a partir de um servidor DHCP. Se não conseguir comunicar com o servidor DHCP porque este está indisponível ou porque configurou incorrectamente as definições de segurança de rede sem fios, a impressora/dispositivo multifunções mudará para um endereço IP baseado em APIPA, dentro do intervalo 169.254.0.1 a 169.254.255.254.
