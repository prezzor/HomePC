Configurazione del certificato.
Certificato client
Selezionare il certificato client per l'autenticazione EAP-TLS. Tenere presente che occorre innanzitutto impostare il Certificato nella pagina "Configura certificato" dalla pagina Web interna (EWS) della periferica in uso. (Quando viene utilizzato un metodo di autenticazione non occorre selezionare il certificato client ad eccezione di EAP-TLS.)
Verifica del certificato server
Selezionare il metodo di verifica del certificato server.(Quando viene utilizzato LEAP non occorre selezionare il metodo di verifica.)
  • Nessuna verifica
    Il certificato server viene considerato attendibile senza verifica.

  • Cert. CA
    Il certificato server emesso da CA viene verificato dal certificato CA installato in questa periferica.Tenere presente che occorre innanzitutto impostare il Certificato nella pagina "Configura certificato" dalla pagina Web interna (EWS) della periferica in uso.

  • Cert. CA + ID server
    Il certificato server (emesso da CA utilizzando il certificato CA installato nell'apparecchiatura) e il nome comune (ID server) vengono verificati.Tenere presente che occorre innanzitutto impostare il Certificato nella pagina "Configura certificato" dalla pagina Web interna (EWS) della periferica in uso.

ID server
Immettere l'ID server per la verifica del certificato server.L'ID server corrisponde al nome comune del certificato server.

Configura certificato
Se si utilizza un certificato client o un certificato CA, occorre innanzitutto impostare il Certificato nella pagina "Configura certificato" dalla pagina Web interna (EWS) della periferica in uso.
