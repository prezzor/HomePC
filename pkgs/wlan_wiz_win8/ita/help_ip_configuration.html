Configurazione indirizzo IP
È necessario configurare la periferica multifunzione o la stampante wireless per l'utilizzo di un indirizzo IP. Sono disponibili due opzioni:

Ottieni automaticamente indirizzo IP. Utilizzare questa opzione se la rete utilizza un server DHCP.

Imposta indirizzo IP statico. Utilizzare questa opzione se la rete non utilizza un server DHCP e si desidera immettere manualmente le impostazioni dell'indirizzo IP e della subnet mask.

Se non si sa quale opzione selezionare, consultare l'amministratore del sistema.

Nota: se si seleziona l'opzione “Ottieni automaticamente indirizzo IP”, la periferica multifunzione o la stampante tenterà di acquisire un indirizzo IP da un server DHCP. Se non è in grado di comunicare con il server DHCP perché quest'ultimo non è disponibile o perché le impostazioni di protezione sono state configurate in modo errato, la periferica multifunzione o la stampante utilizzerà un indirizzo IP basato su APIPA nell'intervallo compreso tra 169.254.0.1 e 169.254.255.254.
