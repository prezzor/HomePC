Description of the Problem:
1. The wireless printer/multifunction device is unable to connect to the wireless network.

Solution:
SSID
Please make sure that you have entered the SSID information, also referred to as network name, correctly when configuring the wireless printer/multifunction device. Please also remember that the SSID information is case sensitive. For example, if your SSID is "Wireless Network" but you enter "wireless network" when configuring the printer/multifunction wireless device, the machine will not be able to connect to the network as the W and the N or "Wireless Network" are not capital letters.

PASSKEY
Please also make sure that you have entered the correct WEP/TKIP passkey when configuring the wireless printer/multifunction device. Enter the correct passkey making sure that you enter the correct number of characters depending on the passkey originally used when configuring the wireless network. In both cases, if you have entered the incorrect information we recommend that you factory reset the wireless settings on the printer/multifunction device and restart the Wireless Wizard software. For more information on how to reset the device wireless settings, please refer to the documentation supplied with the wireless printer/multifunction device.

Description of the Problem:
2. When searching for wireless networks, the Wizard did not find the wireless network.

Solution:
If the wireless printing device is out of range of your wireless network move it closer to the Wireless Access Point.

If the wireless printing device is within range of your wireless network check to see if the Wireless Access Point is not hiding its SSID network name. If it is hiding the SSID network name, you must manually enter it during the configuration of your wireless printing device. If you are unsure of how to check this please consult your systems administrator, Internet Provider or manufacturer of your Wireless Access Point.

(Only applicable to users who are not using a USB cable to configure their wireless printing device). If you are using a personal firewall, the firewall may prevent the Wizard software from finding wireless networks. In which case, you may need to temporarily disable the personal firewall software and try to rediscover the wireless network. Please note: as your computer will not longer be protected by the firewall, you may wish to disable Internet access by disconnecting your Access Point from the Internet.

Description of the Problem:
3. The signal strength is low or sometimes there is no signal strength.

Solution:
Wireless devices affected by the environment that they are in and the signal strength can be altered by the furniture, walls or floors in-between the wireless printer/multifunction device and the Access Point that it is trying to communicate with. If you're experiencing signal strength problems try re-positioning the wireless printer/multifunction device closer to the access point to improve signal strength.

Description of the Problem:
4. (Only applicable to users who are not using a USB or LAN cable to configure their wireless printing device). I made a mistake when configuring the wireless settings, but when I re-run the Wireless Wizard software to enter the correct settings, the Wizard does not find any devices.

Solution:
It's possible that the Wizard software is not able to find your wireless printer/multifunction device because your computer and printer/multifunction wireless settings are now totally different. Try re-setting the wireless settings on the printer/multifunction device to their factory settings and re-start the wizard.

For more information on how to factory reset the printer/multifunction wireless settings, please refer to your user guide.

Description of the Problem:
5. The Wireless Wizard software refers to an access point. But I am not using an access point, I am using a wireless router or a switch to connect to my wireless network.

Solution:
We have used the general term "access point" to keep things simple. Please substitute access point with the appropriate name.

Description of the Problem:
6. I don't know which wireless network settings to use, what should I do?

Solution:
If you incorrectly configure the wireless settings of the printer/multifunction device it will not work correctly. You must speak to the system administrator or someone who understands which wireless network settings to you.

Description of the Problem:
7. I'm trying to use the Secure Easy SetupTM capability of my access point, but it doesn't work.

Solution:
Please make sure that you have correctly enabled the Secure Easy SetupTM option on the access point and the wireless printer/multifunction device. Please refer to the user guide supplied with the printer/multifunction device for wireless setting information.

Description of the Problem:
8. I've configured the device to use a DHCP server. However, the printer/multifunction device is not receiving an IP address from the DHCP server. It's using a 169.254.x.x IP address.

Solution:
This suggests that the wireless printer/multifunction device is not able to access the wireless network and can not obtain an IP address from the DHCP server. Please check that you have correctly entered the wireless network settings on your wireless printer/multifunction machine and also please check that the access point is configured to allow the device to access the wireless network.
